Troubleshooting

If diagrams don't render, try:

  1. Refresh the page (F5 or Cmd+R)
  2. Check your internet connection (Mermaid loads from CDN)
  3. Open browser console (F12) to check for errors
  4. Try opening in a different browser (Chrome, Firefox, Edge)
  5. Wait a few seconds for diagrams to load

Alternative: Visit mermaid.live and copy-paste the Mermaid code from presentation-infographics.md
